====================================
Export Pipeline to Specific Langague
====================================

.. default-domain:: mongodb

*New in version 1.15.0*

You can use the :ref:`aggregation pipeline builder
<compass-agg-builder>` to export your finished pipeline to one of the
supported languages; Java, Node, C#, and Python 3. This feature
facilitates formatting and exporting pipelines created in |compass| for
use in your applications.

Procedure
---------

1. After constructing an aggregation pipeline, click the
   :guilabel:`...` button to the right of the
   :guilabel:`Save Pipeline` button.

   .. note::

      For instructions on creating an aggregation pipeline in
      |compass|, see :ref:`Aggregation Pipeline Builder
      <compass-agg-builder>`.

#. Select :guilabel:`Export to Language`:

   .. figure:: /images/compass/agg-builder-export-dropdown.png
      :figwidth: 720px
      :alt: Aggregation Builder export dropdown

#. In the :guilabel:`Export Pipeline To:` dropdown, select your desired
   language:

   .. figure:: /images/compass/agg-builder-export-lang-select.png
      :figwidth: 720px
      :alt: Aggregation Builder language select

      The :guilabel:`My Pipeline` pane on the left displays your
      pipeline in :manual:`mongo shell <mongo>` syntax.

      The pane to the right displays your pipeline in the language
      selected.

#. *(Optional)*: Check the :guilabel:`Include Import Statements` option
   to include the required import statements for the language selected.

#. Click the :guilabel:`Copy` button at the top-right of the pipeline
   to copy the pipeline for the selected language to your clipboard.
   You can now integrate and execute your created pipeline in your
   application.

#. Click :guilabel:`Close` to return to the aggregation pipeline
   builder.

.. seealso::

   - :ref:`Aggregation Pipeline Builder <compass-agg-builder>`
