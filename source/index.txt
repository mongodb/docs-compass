What is |compass-short|?
------------------------

|compass| is the :abbr:`GUI (Graphical User Interface)` for MongoDB.
|compass-short| allows you to analyze and understand the contents of
your data without formal knowledge of MongoDB
:manual:`query syntax </tutorial/query-documents>`. In addition to
exploring your data in a visual environment, you can also use
|compass-short| to :ref:`optimize query performance <explain-plans>`,
:ref:`manage indexes <compass-indexes>`, and implement
:ref:`document validation <validation>`.

.. _compass-and-community-eds:

What Different Editions of |compass-short| Are Available?
-------------------------------------------------------------

|compass-short| is offered in the following four editions:

.. tabs:: 

   tabs:

     - id: compass-free
       name: Compass Community
       content: |

         Compass Community is a version of MongoDB Compass that
         supports a subset of functionality appropriate for
         developing with MongoDB. Compass Community is available
         at no cost to all users.

         Compass Community supports features such as viewing 
         cluster data and interacting via CRUD functionality. For
         a complete list of features, see
         :ref:`compass-feature-table`.

     - id: compass
       name: Compass
       content: |

         Compass is the full version of MongoDB Compass, with 
         complete availability to all features and capabilities. 
         Compass is appropriate for both development and management
         of MongoDB clusters.

         Compass is only available as a subscription. For more
         information on MongoDB Compass subscriptions, see <link>.  For
         a complete list of features, see :ref:`compass-feature-table`.

     - id: compass-readonly
       name: Compass Readonly
       content: |

         Compass Readonly is limited to only read operations, with
         all write or deletion capabilities disabled. Compass Readonly
         is appropriate for supporting data analytics or data
         scientists.

         Compass Readonly is only available as a subscription. For more
         information on MongoDB Compass subscriptions, see <link>. For
         a complete list of features, see :ref:`compass-feature-table`.

     - id: compass-isolated
       name: Compass Isolated
       content: |

         Compass Isolated only allows TLS-encrypted outbound connections
         to the target MongoDB cluster. All other outbound network
         connections are restricted. Compass Isolated cannot
         support features that require outbound connections to
         third-party services, such as automatic updates or crash
         reporting. Compass Isolated is appropriate for organizations
         with high security and privacy requirements.

         Compass Isolated is only available as a subscription. For more
         information on MongoDB Compass subscriptions, see <link>. For
         a complete list of features, see :ref:`compass-feature-table`.
