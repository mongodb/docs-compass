What is |compass-short|?
------------------------

|compass| is the :abbr:`GUI (Graphical User Interface)` for MongoDB.
|compass-short| allows you to analyze and understand the contents of
your data without formal knowledge of MongoDB's
:manual:`query language </tutorial/query-documents>`. |compass-short|
also enables you to :ref:`optimize query performance <explain-plans>`,
:ref:`manage indexes <compass-indexes>`, and implement
:ref:`document validation <validation>` from within a convenient user interface.

.. _compass-and-community-eds:

What Different Editions of |compass-short| Are Available?
-------------------------------------------------------------

|compass-short| is offered in four editions:

.. list-table::
   :header-rows: 1
   :widths: 25 70 5

   * - Edition
     - Description
     - Subscription Required

   * - Compass
     - Full version of |compass|, with all features and
       capabilities. |compass-short| is available as part of 
       a MongoDB subscription.
     - :icon:`check-circle`

   * - Compass Community
     - This version is for developing with MongoDB and includes a
       subset of the features of Compass.
     -

   * - Compass Readonly
     - This version is limited strictly to read operations, with all
       write and delete capabilities removed. Available as part of a 
       MongoDB subscription.
     - :icon:`check-circle`

   * - Compass Isolated
     - This version disables all network connections except the
       connection to the MongoDB instance.
       Available as part of a MongoDB subscription.
     - :icon:`check-circle`
